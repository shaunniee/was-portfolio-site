Static Portfolio Site v2 (this site)

S3 • CloudFront • WAF • Route 53 • ACM • CodePipeline • CodeBuild

Static HTML/CSS site hosted in a private S3 bucket, fronted by a CloudFront distribution with WAF, custom domain and TLS managed by ACM. Deployed automatically from GitHub on every push to main.

  • Private S3 origin accessed only via CloudFront Origin Access Control (OAC).
  • AWS WAF Web ACL with managed rules and rate limiting in front of CloudFront.
  • CloudFront & WAF logs shipped to dedicated S3 buckets for analysis.
  • CodePipeline & CodeBuild sync content to S3 and invalidate CloudFront caches.

How this site is deployed

High-level summary

Code lives in a GitHub repo. A CodePipeline in eu-west-1 listens for pushes to main and triggers CodeBuild.

CodeBuild uses a buildspec.yml file to sync artifacts to a private S3 bucket and then creates a CloudFront invalidation so new content goes live immediately.

CloudFront sits in front of S3 with an Origin Access Control, AWS WAF Web ACL, and an ACM certificate for the custom domain.
